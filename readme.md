# Test

I test all things in `~/test`, create a folder there like [jazz](https://github.com/nikitavoloboev/jazz-test) and open it as [cursor workspace](https://code.visualstudio.com/docs/editor/workspaces) together with fork of tech I test against (i.e. [jazz](https://github.com/nikitavoloboev/jazz)). This lets me test things fast and upstream changes with PRs where useful.

All test repos end with `-test` suffix and can be seen [here](https://github.com/nikitavoloboev?tab=repositories&q=-test&type=source).
